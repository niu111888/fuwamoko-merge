# ふわもこタイピング 3カ国語 — 仕様まとめ

`typing.html`（1ファイル完結・Canvas 2D・依存なし）
公開URL: https://niu111888.github.io/fuwamoko-merge/typing.html

英語・中国語・日本語を、いろんな遊び方で練習できるタイピングゲーム。かわいいパステル調で、スマホ・PC両対応（キーボードはPC推奨）。

---

## 画面の流れ

```
① 言語 → ② コース → ③ あそびかた(＋オプション) → ▶ プレイ → 結果
                                                        └ 📊 にがてキー カルテ（① からも開ける）
```

- 各画面に「← もどる」、プレイ中は左上に **🏠**（ゲーム一覧へ）と右上に「⏸ やめる」。
- 記録・設定はすべてブラウザに自動保存（localStorage）。

---

## ① 言語
**English / 中文 / 日本語 / ぜんぶMIX**（MIXは単語だけを言語ミックス）。
この画面から **📊 にがてキー カルテ**（キーボードの `K`）も開ける。

## ② コース（言語ごと）
- **たんご**（3段階）：やさしい / ふつう / むずかしい（速度・ライフ・単語の長さが変化）
- **ぶんしょう**：
  - 英：TOEIC かんたん(~400) / ふつう(~600) / むずかしい(~800)、ビジネス英語、小説・物語、**映画の名セリフ**、**名作の長文**
  - 中：HSK 1-2 / 3-4 / 5-6、ビジネス中国語、小説・物語、**映画の名セリフ**、**名作の長文**
  - 日：やさしい / ふつう / ビジネス / 小説・物語、**えいがの名セリフ**、**名作の長文**

## ③ あそびかた（5種類）
| モード | 内容 | ライフ | 時間 |
|--------|------|--------|------|
| 🎮 おちもの | 上から落ちる語を打つ | あり | 任意 |
| 🌊 ながれもの | 横に流れる語を打つ | あり | 任意 |
| ⏱ タイムアタック | 制限時間で何問打てるか | なし | 固定 |
| ✍️ れんしゅう | 自分のペース・失敗なし | なし | 任意 |
| 🎧 きいて打つ | 音声だけ→書き取り→字幕が出る | なし | 任意 |

**オプション（あそびかた画面のトグル）**
- ⏱ じかん：∞ / 1分 / 2分 / 3分（おちもの・ながれものは「ライフ＋時間」、タイムアタックは時間で採点）
- ローマ字ガイド：あり / なし（「なし」は かな・漢字だけ見て打つ挑戦モード。英語は単語自体が対象なので常に表示）

## 入力エンジン（言語ごと）

- **日本語＝ローマ字音節エンジン**：検証済みの正しいローマ字を音節に分解し、**複数の打ち方を受理**。
  - し=`si`/`shi`、しゅ=`shu`/`syu`/`shyu`/`sixyu`/`shixyu`、ち=`chi`/`ti`、つ=`tsu`/`tu`、ふ=`fu`/`hu`、じ/ぢ/ず/づ、ヴ、拗音（分割入力含む）、促音っ（子音重ね/`xtu`等）、「ん」（`n`/`nn`・な行や母音前も正しく判定）。
  - 助詞「は=wa / へ=e / を=o」も正しい（`typed`を土台にしているため）。**スペースは打たなくてよい**（語間はすき間だけ）。
- **中国語＝ピンイン**：**スペース無しで連続入力OK**（区切りは自動スキップ・すき間表示）。
- **英語＝単語そのまま**：単語区切りのスペースは打つ（自然なすき間で表示）。

## 🎧 きいて打つ の詳細
- Web Speech API（音声合成）で読み上げ → 本文は隠れて「…」だけ → 聞き取って打つと打てた分だけ現れる → 完成で**映画字幕風に全文＋作品名**が出る → 次へ。
- 🔊 もういちど / 👀 こたえ（ギブアップ）/ 🎤 こえ切替 / スピード（🐢ゆっくり・ふつう・🐇はやい）。
- 声は端末内の**自然な声を自動選択**（Samantha/Kyoko/Ting-Ting等を優先、余興ボイスは回避）。
- 映画・長文・きいて打つコースは**ダークシアター風UI**（黒背景＋レターボックス＋金色字幕）。

## 📊 にがてキー カルテ
- 打ち間違えたキーを記録（**プレイをまたいで累計**）。
- **QWERTYヒートマップ**（赤いほど苦手）＋**指ごとのミス集計**（左小〜右小の8本）。
- 「▶ にがて集中れんしゅう」＝苦手キーを重み付けした練習を生成。「🗑 リセット」で初期化。

## 結果画面
スコア / ベスト / クリア数 / 正確率 / WPM / 最大コンボ ＋ **にがてキー（累計TOP）**をキーキャップ風チップで表示。

---

## データと著作権
- 文章（ローマ字・ピンイン）は **多エージェントのワークフロー（生成→検証→敵対的再チェック）** で正確性を確認して収録。
- **映画の名セリフ**＝短い有名フレーズのみ（短い語句は一般に著作権保護対象外）。実際の映像・音声は使わず、読み上げはTTS。
- **名作の長文**＝パブリックドメイン作品（英：シェイクスピア/ディケンズ/リンカーン/メルヴィル等、日：漱石/太宰/芥川/賢治、中：李白/論語/道徳経/唐詩等）。
- 収録数の目安：単語 各言語≈20、映画セリフ 英16/中14/日13、長文 英7/中6/日5、レベル/ジャンル文 各≈8。

## 技術メモ
- 1つのHTMLファイルにHTML+CSS+JSを内包、外部依存なし。全画面レスポンシブ。
- 保存キー：`typingBest`（最高スコア）/ `typingKeyStats`（苦手キー累計）/ `typingRomaji`（ガイド表示）/ `typingTime`（制限時間）/ `typingVoice`・`typingTtsRate`（音声設定）。
- モバイルは画面外の隠し `<input>` でソフトキーボードの入力を受ける。PCは物理キー。
- 描画は等幅をやめて**自然な文字幅**、入力済みは色を変えてハイライト。
